（持续更新）从jekyll到hexo的迁移旅途

2020.05.11 subrige  阅读

动机

在为jekyll美化的过程中，发现有许多插件都是基于hexo，这些插件移植到jekyll的过程十分令人喷粪。加上hexo的部署过程更符合我的使用习惯，就想着什么时候能把我的博客系统搞到hexo上。

选择主题：Butterfly

开创博客框架

打开终端，先安装hexo支持，噼里啪啦：

npm install hexo-cli -g

然后打开你想要放置hexo博客主文件夹的文件夹。

cd ./yourfolder

接下来创建一个hexo模板。

hexo init yourawseomehexoblog

然后进入这个文件夹，可以看到很多东西。先别着急，还没配置完成，继续噼里啪啦：
